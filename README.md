# Meteostation
**VUT FIT 2024/25 - ITS**  

Simple meteostation implementation for microcontroller ESP32 utilizing sensor and OLED display over I2C bus to measure and display information about currect temprerature and humidity. Also allows to connect to wifi network and send measurments using MQTT protocol. Implemented using ESP-IDF framework.  
More info in documentation.
